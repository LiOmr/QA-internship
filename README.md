# QA Task 1, 2.2 (E2E)

- `TESTCASES.md` — тест-кейсы для критических пользовательских сценариев
- `tests/` — автотесты (Playwright, TypeScript)
- `Task1.md` — первое задание 

Тестируется: https://avito-tech-internship-psi.vercel.app


## Установка

1) Зависимости:

```bash
npm ci
```

2) Playwright:

```bash
npx playwright install
```

## Запуск автотестов

```bash
npx playwright test
```


## Автотесты реализуют пользовательские сценарии:

- TC-01 Создание задачи (позитивный)
- TC-02 Создание задачи без названия (негативный)
- TC-03 Открытие карточки задачи
- TC-04 Поиск задачи (позитивный)
- TC-05 Поиск задачи (негативный)
- TC-06 Переход на доску проекта (из раздела «Проекты»)
